# WA ENC Decryptor

Microserviço para **descriptografar mídias .enc do WhatsApp** (imagem/áudio) via HTTP.
- **GET** e **POST**
- Retorna **binário** ou **base64**
- Compatível com **n8n**, **EasyPanel** e **Compose Beta**

## Endpoints
- GET /decrypt
- POST /decrypt
- POST /decrypt-batch
